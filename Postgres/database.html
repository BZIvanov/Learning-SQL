SQL Sublanguages

DDL - Data Definition Language

DDL consists of the following operations: CREATE, ALTER and DROP

DCL - Data Control Language

DCL is related to security. For example what an user is allowed or disallowed. DCL consists of the following operations: GRANT and REVOKE

DML - Data Manipulation Language

DML consists of INSERT, UPDATE and DELETE

DQL - Data Query Language

DQL consists of SELECT

DTL - Data Transaction Language

DTL consists of BEGIN | START, WORK | TRANSACTION, SAVEPOINT, COMMIT, ROLLBACK
